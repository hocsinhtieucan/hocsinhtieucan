Hướng dẫn đưa lên GitHub Pages

  1. Xuất file JSON bằng cách nhấn nút "Xuất JSON"
  2. Tạo repository trên GitHub
  3. Upload toàn bộ code HTML, CSS, JS lên repository
  4. Tạo thư mục "data" và upload file posts.json đã xuất vào thư mục đó
  5. Trong repository settings, bật GitHub Pages (Settings -> Pages)
  6. Mỗi khi thêm bài mới, xuất JSON và cập nhật file trong thư mục data
